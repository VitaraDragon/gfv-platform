Questa cartella è una **copia** di `docs-sviluppo/guida-app/` usata da **Tony** (assistente IA) per rispondere alle domande degli utenti sull’app.

- **Origine:** `docs-sviluppo/guida-app/` (Core, Terreni, Lavori, Vigneto, Frutteto, Magazzino, Conto terzi, Intersezioni).

**Aggiornare la guida:** quando modifichi i file in `docs-sviluppo/guida-app/`, copia di nuovo i file qui (stessi nomi e struttura) per avere Tony sempre allineato alla documentazione.
