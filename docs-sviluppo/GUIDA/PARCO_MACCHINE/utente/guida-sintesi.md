# Parco Macchine — sintesi per Tony

Modulo **Parco macchine** attivo nell’**abbonamento** (scheda moduli per l’amministratore). Linguaggio allineato alla guida utente.

## In sintesi

- **Moduli** → **Parco Macchine** → hub con statistiche e **azioni rapide** (trattori, attrezzi, flotta, gestione macchine, scadenze, officina/guasti). Ingresso anche da **I miei accessi** o alert **Richiede attenzione** (scadenze/guasti); riquadro in fondo dashboard solo **senza** Manodopera. Panoramica flotta: conteggio mezzi e **km totali** (non ore).
- **Liste rapide** (Trattori / Attrezzi / Flotta): solo **creazione** rapida (**Nuovo** → campi minimi → **Salva**); **modifica / elimina / stato completo** da **Gestione macchine**.
- **Trattori:** nome + **CV** obbligatori alla creazione rapida; colonna **Ore** e **Stato** (Disponibile, In uso, Manutenzione, Guasto); manutenzione per **ore** e/o **data**; revisione e assicurazione opzionali.
- **Attrezzi:** nome, **categoria** (elenco), **CV minimi** obbligatori — servono per **compatibilità** con il trattore (CV trattore ≥ CV minimi attrezzo); contatore **ore**; Stato spesso Disponibile / In uso.
- **Flotta** (Automezzo / Veicolo / Furgone): nome + **tipo**; elenco con colonna **Km** (non ore). Tagliando e contatore in **chilometri**; revisione e assicurazione per data in Gestione macchine.
- **Gestione macchine:** filtri stato/tipo (incluso **Flotta**)/categoria/attive; **Nuova macchina** — **Trattore**, **Attrezzo** o **Mezzo aziendale**; form completo — **Stato** (incluso **Dismesso**), CV trattore o CV minimi attrezzo, marca/modello/targa/telaio; **ore** + costo/h per agricoli; **km** + prossimo tagliando km per flotta; prossima manutenzione (data), revisione, assicurazione; note, storico guasti; righe con **Modifica** / **Elimina**.
- **Scadenze:** filtro Tutti / Solo scaduti; tipi riga — Manutenzione (data), **Manutenzione (ore)** (trattori/attrezzi), **Tagliando (km)** (flotta), **Revisione**, **Assicurazione**; rinnovo con **nuova data**, **nuove ore** limite o **nuovo limite km**; indicatori colore (rosso/giallo/verde).
- **Guasti:** filtro Solo aperti / Tutti; **Segnala guasto** → pagina dedicata; **Segna risolto** + note; può aggiornare stato macchina collegata.
- **Diario attività:** trattore / attrezzo / ore macchina **opzionali** (solo mezzi agricoli); combinazioni rispettano CV. La flotta stradale **non** usa ore nel Diario.

Senza modulo **non** dare istruzioni come se il parco esistesse (né voce in **Moduli**).
