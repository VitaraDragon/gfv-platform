Guide utente (GUIDA)

Ogni voce apre la versione dedicata (testo caricato dai file guida in markdown).

  • 📗 Core — dati azienda, terreni, diario, ruoli GUIDA
  • 🚜 Parco Macchine GUIDA
  • 🍇 Vigneto GUIDA
  • 🍎 Frutteto GUIDA
  • 📦 Magazzino GUIDA
  • 🤝 Conto Terzi GUIDA
  • 👷 Manodopera GUIDA
  • 🤖 Tony (modulo assistente) GUIDA
Da dove iniziare
  • Core se devi capire ruoli, piani e voci comuni.
  • Poi il modulo che usi ogni giorno.
  • Tony se ti serve capire chat, voce e piano.

Benvenuto

Qui vedi solo le guide dei moduli attivi sul tuo abbonamento (apertura dalla dashboard). Core resta sempre disponibile.

Sei in vista campo (operaio o caposquadra): è disponibile solo la guida Manodopera per il tuo ruolo.

Usa la voce Manodopera nel menu a sinistra per aprire la guida del tuo ruolo.

Scegli una guida dal menu a sinistra: si apre una pagina dedicata, comoda da leggere e da stampare.

Le guide sono allineate al materiale GUIDA del progetto (stesso testo che l’assistente Tony può usare come riferimento).

Suggerimento: in locale usa un piccolo server (es. dalla cartella del progetto npx serve) così il browser può caricare i file markdown; con doppio clic sui file HTML alcuni browser bloccano il caricamento.
